# Minimal Pi Clock

The Minimal Pi Clock is
[one of my machines](https://github.com/richelbilderbeek/Machines)
that uses (the programmable chip of an) Arduino.

It's an Arduino project for a clock that beeps at pi o'clock PM
(i.e. 3:14 PM or 15:14). It can easily be made to beep at 3:14 AM
for the purists :-)
